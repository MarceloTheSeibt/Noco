Este repositório possui código-fonte do programa e os lançamentos beta e final.
Para ter acesso ao código-fonte da versão final, altere o branch.

Como baixar o jogo:

1 - Para baixar o jogo, selecione o lançamento (release) Final ou Beta na lateral direita da tela do repositório. 

2 - Após ter selecionado a versão, clique no arquivo .rar e baixe-o. 

3 - Extraia o arquivo .rar em qualquer local do seu computador. 

4 - Abra a pasta extraída e abra o arquivo .exe.
